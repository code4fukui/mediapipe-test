# mediapipe-test

MediaPipeを使ったさまざまな機能のテスト集です。顔検出、ポーズ推定、手検出などのデモを提供しています。

## デモ

以下のデモを試すことができます:

- [全身 - Holistic (Face Mesh + Hands + Pose + Selfie Segmentation)](https://code4fukui.github.io/mediapipe-test/holistic.html)
- [顔 - Face Detection](https://code4fukui.github.io/mediapipe-test/facedetection.html)
- [顔メッシュ - Face Mesh](https://code4fukui.github.io/mediapipe-test/facemesh.html)
- [顔メッシュ（詳細） - Face Mesh mesh test](https://code4fukui.github.io/mediapipe-test/facemesh-mesh.html)
- [手 - Hands](https://code4fukui.github.io/mediapipe-test/hands.html)
- [背景分離 - Selfie Segmentation](https://code4fukui.github.io/mediapipe-test/selfiesegmentation.html)
- [体 - Pose](https://code4fukui.github.io/mediapipe-test/pose.html)
- [物体検出 - Object Detection (Coco-SSD)](https://code4fukui.github.io/mediapipe-test/objectdetection.html)
- [体複数認識（物体検出を使用） - Pose/Coco-SSD (PoseMulti)](https://code4fukui.github.io/mediapipe-test/pose-multi.html)

## 機能

- 顔、手、ポーズなどさまざまな特徴の検出
- 背景分離
- 物体検出

## 必要環境

特に必要ありません。ブラウザで動作します。

## 使い方

ブラウザで上記のデモページを開いてください。
